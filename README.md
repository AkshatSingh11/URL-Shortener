# URL Shortener (Node.js + Redis)

A scalable URL shortener built with Node.js, MongoDB, and Redis caching.

## 🚀 Features

* Shorten long URLs
* Fast redirection
* Redis caching for performance
* Rate limiting to prevent abuse

## 🧠 Tech Stack

* Node.js, Express
* MongoDB
* Redis

## ⚡ Performance

* Reduced DB hits using caching
* Faster redirect response time

## 📦 Setup

npm install
npm start

## 📌 API

POST /shorten
GET /:shortId
